📖 How Codes Work

1. Generate a batch of codes (e.g. IK2-001 to IK2-5000)

2. Print all 5,000 QR stickers identically — same QR, same design

3. Distribute stickers to field agents with a separate list of their assigned codes

4. Agent arrives at location → writes assigned code on the sticker's blank space → registers it via the Field Agent Portal

5. Public scans QR → enters code → scan is attributed to that location

6. Admin sees scan data per location on this dashboard
